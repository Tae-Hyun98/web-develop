table 만들기

html에서 표를 구현할때

  • tr(table row):행
  • td(table data):(행의 내용) 열
  • th(table heading):행, 열의 제목(굵음체, 가운데정렬)
  • 속성 colspan="2" : 2칸의 열병합
  • 속성 rowspan="2" : 2칸의 행병합
